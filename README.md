# 개인용 DC 단축키

DC에서 자주 쓰는 단축키와 유저 정보 표시 기능을 제공하는 개인용 크롬 확장 프로그램입니다.

## 단축키

| 키 | 동작 |
|----|------|
| W | 글쓰기 |
| C | 댓글 영역 이동 (+ 선택적 댓글 새로고침) |
| R | 페이지 새로고침 |
| F | 전체글 이동 |
| G | 개념글 이동 |
| A | 이전 페이지 (최신 방향) |
| S | 다음 페이지 |
| Z | 이전 글 |
| X | 다음 글 |
| 1~9, 0 | 현재 페이지 목록의 1~10번째 글로 이동 |

## 유저 정보 표시

- 글 목록, 글보기, 댓글에서 작성자의 IP/ID를 색상 배지로 표시
- 같은 IP 또는 ID는 동일한 색상으로 표시되어 동일인 식별이 쉬움
- 고닉/반고닉: `[uid]` 배지, 유동닉: `(IP)` 배지

## 설치 방법

1. 압축을 푼 뒤 폴더를 준비합니다.
2. 크롬 주소창에 `chrome://extensions` 를 엽니다.
3. 우측 상단의 `개발자 모드` 를 켭니다.
4. `압축해제된 확장 프로그램을 로드` 를 누릅니다.
5. 이 폴더를 선택합니다.

## 메모

- 한/영 상태와 상관없이 키 위치 기준(`event.code`)으로 동작합니다.
- 입력창에 포커스가 있을 때는 기본적으로 단축키를 무시합니다.
- 설정은 확장 아이콘을 눌러 팝업에서 켜고 끌 수 있습니다.

## 개발 도구

- **Claude Code Opus 4.6** — 기획, 명세서 작성, 코드 리뷰, 아키텍처 설계
- **OpenAI Codex (GPT-5.4)** — 코드 구현, adversarial review, 디버깅
